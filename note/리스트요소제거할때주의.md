### 리스트에서 특정 요소를 제거할 때

리스트 슬라이싱을 사용하지 않고
pop이나 del을 이용하여 제거


프로그래머스 햄버거 만들기 문제 풀이 예시

_ex1) del 사용_

```python
def solution(ingredient):
    s = []
    cnt = 0
    for i in ingredient:
        s.append(i)
        if s[-4:] == [1, 2, 3, 1]:
            cnt += 1
            for _ in range(4):
                s.pop()
    return cnt
 ```
 
 _ex2) pop 사용_
